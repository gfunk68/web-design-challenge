Weather Data

This project is analyzing how weather changes based on latitude. This data is from the OpenWeatherMap API and has over 500 cities. We will be looking at a specific date of January 5th 2017.

The dataset was then analyzed with python and the results were plotted into various charts and graphs using Matplotlib. We look at max tempuratures, humidty, cloudiness, and wind speed. Please see links to the visualizations below.
